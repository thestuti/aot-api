Attack On Titan Quotes API ⚔️

A free API to retrive some quotes of Attack On Titan

Star
❯ Get a random quote
GET /quotes/random
❯ Get specified number of random quotes,
has to be less than or equal to existing number of quotes
GET /quotes/random?count=<integer>

❯ All quotes

GET /quotes/
❯ Get specified number of quotes,
has to be less than or equal to existing number of quotes
GET /quotes?count=<integer>

How to contribute


If you want to suggest a quote from Attack On Titan, go to the GitHub repository, fork the project, add your quote to the quotes.js file and just do a pull request!
